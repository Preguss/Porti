# Portfólio - Preguss

Este é o meu portfólio pessoal, hospedado no GitHub Pages.

## Funcionalidades
- Página inicial com sobre, habilidades, projetos e experiência.
- Blog pessoal com posts dinâmicos armazenados no Firebase.

## Como Configurar o Firebase
1. Acesse [Firebase Console](https://console.firebase.google.com).
2. Crie um novo projeto.
3. Ative o Firestore Database.
4. Vá em Project Settings > General > Your apps > Add app (Web).
5. Copie as credenciais (apiKey, authDomain, etc.) e substitua no `blog.html` no `firebaseConfig`.

## Deploy
- O site é hospedado via GitHub Pages.
- Para atualizar, faça push para a branch main.

## Tecnologias
- HTML, CSS, JavaScript
